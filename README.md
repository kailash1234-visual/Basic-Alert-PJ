# 📱 Alert System

A Python script that runs on Android using Termux.
Get notified when you've been on your phone too long and never miss gym time!

---

## ✅ What it does
- 🔔 Alerts every **30 minutes** of continuous phone use
- 💪 **Gym Time** alert at exactly **4:00 PM** every day
- ❌ Skips **Sunday** for gym alert
- 📳 Vibrates your phone for both alerts
- 📬 Sends real phone notifications

---

## 📲 Requirements
- Android phone
- Termux app (from [f-droid.org](https://f-droid.org))
- Termux:API app (from Play Store)
